SNORKYTHEBEARD SPRITE TRACKER WEB APP

This folder is ready to host as a small installable web app.

FILES
- index.html: the app
- manifest.webmanifest: Home Screen/app information
- service-worker.js: offline support
- icons/: Home Screen icons

EASIEST DEPLOYMENT
1. Sign in to Netlify.
2. Open Netlify Drop / manual deploy.
3. Upload this entire folder, not only index.html.
4. Netlify gives you a web address.
5. Open that address in Safari on iPhone.
6. Tap Share > Add to Home Screen.

Progress is stored in that browser on that phone. Clearing Safari website data can erase it.
